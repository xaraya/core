Requirements for running Xaraya 2.3.0

This page summarizes the system you need to run Xaraya 2.3.0. Xaraya will run on most operating systems, and its installer will guide you through the steps required for a correct installation of Xaraya. However, some things must be in place before the installer can properly function. These are listed below.

If you have not already done so please read the file readme.html in this directory for information on how to configure and install Xaraya. For more support resources click on the first link in the header above.

List of basic requirements

  • A web server that can run the PHP programming language, such as Apache or IIS.
  • A relationsal database Xaraya supports, such as MySQL, PostgeSQL or SQLite.
  • PHP version 5.3.0 or higher installed
  • The PHP XML extension installed
  • The PHP XSL extension installed

If you are seeing this page after starting the installer, it means one or more of the PHP requirements above have not been met. If so please consult the documentation at the PHP site for information on how to install PHP or PHP extensions on your operating system.

Once the requirements above are met you can run the installer and will not see this page.
